### Map 组件

> 设计原理

- 基于高德地图 web 端 SDK 定制，通过 script-loader 来动态加载远程的 CDN 完成组件的初始化
- 借由暴露实例化的地图组件来完成控制权的转让，由父组件来控制组件本身，地图组件本身只负责，接受数据和处理
